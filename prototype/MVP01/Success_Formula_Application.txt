파일명: Success_Formula_Application.py
1. 프레임 추출: OpenCV를 사용하여 영상을 1초 단위로 쪼개기(리소스 최소화)
2. 멀티모달 대조 분석: 추출된 프레임과 시장 데이터를 GPT-4o에 전달하여 분석 
3. 결과 시각화: 분석된 점수와 추천 프레임을 화면에 출력

개선 사항
- 데이터 경량화 (교수님 피드백 반영): 원본 영상 전체를 보내는 대신, fps 값을 이용해 1초에 1장(1 FPS)만 추출하여 API 토큰 소모를 최소화
- 멀티모달 대조 분석: 시장 데이터(success_formula_text)를 참고 정보로 함께 전달하여, 단순 분석이 아닌 '시장 트렌드와 내 영상의 일치도'를 찾아내도록 설계
- 시각적 피드백: matplotlib을 활용해 분석 대상이 된 주요 프레임들을 시각화하여 보여줌으로써, 크리에이터가 어떤 장면을 기준으로 분석이 되었는지 직관적으로 알 수 있게 설계
