1. 파일명: Collect_Popular_Video_data.py (및 AI 통합 로직)

2. 주요 목적
해당 카테고리의 유튜브 시장의 실시간 트렌드(성공 공식)를 데이터 기반으로 분석하기 위한 기초 데이터 수집 및 인사이트 도출.
멀티모달 AI(Vision + NLP)를 활용<-OpenAI API 사용하여 구현

3. 주요 기능 및 작동 원리
데이터 수집 (YouTube Data API v3): 특정 카테고리(예: 'Howto & Style', ID: 26)의 '인기 급상승(Most Popular)' 영상 30개를 타겟팅하여 수집함.
영상의 제목(Title), 조회수(Statistics), 고화질 썸네일 URL(Thumbnail URL) 정보를 확보함.

성공 패턴 분석 (OpenAI GPT-4o Multi-modal):
수집된 30개 영상의 제목 패턴을 분석하여 클릭을 유도하는 '언어적 트렌드'를 파악함.
상위 30개 영상의 썸네일 이미지를 AI가 직접 시각적으로 분석하여 색감, 조명, 구도 등 '시각적 성공 공식'을 도출함.

맞춤형 전략 매칭:
시장의 데이터(Trend)와 사용자가 업로드한 원본 영상(User Content) 사이의 간극을 분석하여 최적의 썸네일 장면과 제목 키워드를 제안함.

4. 핵심 기술적 특징 (Key Features)
지능형 샘플링: 전체 30개 영상을 분석하여 API 비용을 최적화하고 정확도를 높임.
보안 강화: .env 파일을 활용하여 YouTube 및 OpenAI API Key를 안전하게 관리(Environment Variables).
멀티모달 퓨전: 텍스트(제목)와 이미지(썸네일)를 개별적으로 보지 않고, 하나의 통합된 맥락에서 성공 요인을 분석하는 '멀티모달 전략' 채택.
