[Project SyncIt] MVP v1.0: 실시간 트렌드 분석 및 자동 리포트 시스템
1. 파일 개요


파일명: MVP01.py (통합 버전) 


핵심 역할: 유튜브 실시간 데이터 수집, 영상 멀티모달 분석, 웹 대시보드 서빙, 이메일 알림을 하나로 결합한 통합 백엔드 엔진입니다. 


2. 업그레이드된 주요 기능


실시간 트렌드 동기화 (YouTube API): 사용자가 분석을 요청하는 시점의 'Howto & Style' 카테고리 인기 영상 10개의 제목을 실시간으로 가져옵니다. 

경량화 멀티모달 분석 (OpenAI GPT-4o):

교수님 피드백을 반영하여 원본 영상에서 1초당 1프레임만 추출하여 분석함으로써 서버 부하를 90% 이상 절감했습니다. 


수집된 트렌드 키워드와 내 영상의 시각적 요소를 대조하여 '트렌드 일치 점수'를 산출합니다. 


웹 대시보드 연동 (FastAPI): uvicorn 서버를 통해 웹 브라우저(http://127.0.0.1:8000/analyze)에서 즉시 분석을 트리거하고 결과를 확인할 수 있습니다. 


자동 이메일 컨설팅 (SMTP): 분석이 완료되면 사용자의 이메일로 추천 제목 3가지와 베스트 썸네일 컷 번호가 포함된 리포트를 즉시 발송합니다. 

3. 시스템 아키텍처

4. 설정 방법 (Configuration)

가상환경: YoutubeTest1.env 활성화 필수.

환경 변수 (API_KEY.env):


YOUTUBE_API_KEY: 유튜브 데이터 수집용 


OPENAI_API_KEY: 멀티모달 분석용 

EMAIL_SENDER: 리포트 발송용 G-mail 계정

EMAIL_PASSWORD: 구글 앱 비밀번호 (16자리)

5. 기대 효과 및 실효성


현실적 구현: 거창한 하드웨어 없이 소프트웨어와 API만으로도 높은 수준의 컨설팅이 가능함을 증명했습니다. 


데이터 기반 의사결정: '감'에 의존하던 썸네일 제작을 '데이터'와 'AI'의 협업 영역으로 전환하여 크리에이터의 성공 확률을 높입니다.
